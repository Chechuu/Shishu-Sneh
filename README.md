# 👶 Shishu-Sneh Lite (Healthcare)

![App Banner](shishu_sneh_banner_1778808984519.png)

**Shishu-Sneh** is a minimalist, core-focused Android application designed to support new mothers. It acts as a "Digital Elder," providing critical guidance during a baby's first vital year.

## ✨ Premium UI & Features
The app features a calming pastel design with a focus on one-handed operation and clear, glanceable health data.

### 📊 Growth Tracker
Monitor your baby's development with an animated, professional-grade weight chart.
![Growth Tracker](screenshots/growth.jpg)

### 💉 Interactive Immunization
A personalized vaccination timeline that reacts to your baby's birth date. Track progress with a live progress bar.
![Vaccination Page](screenshots/vaccines.jpg)

### 🥗 Care Hub (Feeding & Milestones)
Comprehensive guides for nutrition and developmental milestones organized by age and category.

## 🛠️ Technical Implementation
- **UI Architecture**: Jetpack Compose (Material 3) with Unidirectional Data Flow.
- **Data Persistence**: Room Database for offline-first reliability in rural areas.
- **Charts**: MPAndroidChart with custom Cubic Bezier configurations.
- **Modern Stack**: Kotlin 1.9.22, Gradle 8.5, and Java 21.

## 📦 Getting Started
1. Clone the repository.
2. Open in Android Studio Iguana+.
3. Ensure Gradle JDK is set to **Java 21**.
4. Build and Run on an emulator or physical device.
